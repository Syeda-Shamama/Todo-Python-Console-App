# Todo Console Application

This is a simple, in-memory command-line todo application written in Python.

## Features

- Add, list, update, and delete tasks.
- Mark tasks as complete or incomplete.
- All tasks are stored in memory and are lost when the application exits.

## Requirements

- Python 3.13+
- UV Package Manager

## How to Run

1.  Navigate to the `src` directory.
2.  Run the application using the following command:

    ```bash
    python main.py
    ```

## Commands

- **1. Add task**: Add a new task with a title and description.
- **2. List tasks**: View all your tasks.
- **3. Update task**: Edit the title and description of a task.
- **4. Delete task**: Remove a task from the list.
- **5. Mark complete/incomplete**: Toggle the completion status of a task.
- **6. Exit**: Quit the application.
